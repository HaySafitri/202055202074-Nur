Pengertian HTML

HyperText Markup (HTML) Merupakan suatu bahasa markah yang digunakan untuk membuat halaman WEB, menampilkan berbagai informasi didalam sebuah browser web. HTML dibuat dengan menggunakan aplikasi pengolah kata dan disimpan dalam format ASCII sehingga menjadi halaman web dengan perinth-perintah HTML. HTML merupakan sebuah standar yang digunakan secara luas untuk menampilkan halaman web.pada saat HTML merupakan standar internet yang didefinisikan dan dikendalikan penggunaannya oleh Word Wide Web Consortium(W3C). HTML dibuat oleh Caillau TIM yang diketuai oleh Berners-lee robert ketika bekerja di CERN padatahun 1989.
